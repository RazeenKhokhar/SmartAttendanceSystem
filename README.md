# 🚀 Smart Attendance System v2.0

## Overview
Advanced desktop app for attendance management with **multi-class support**, **beautiful gradients**, **dark mode**, **charts**, **facial recognition**, **QR codes**, **login**, tests.

**Built with**: Java Swing, CSV, JFreeChart, OpenCV, ZXing.

## Features
- ✅ Multi-class students (add/edit/delete)
- ✅ Class-filtered attendance marking
- ✅ Dashboard + theme toggle
- ✅ Charts (attendance trends)
- ✅ Facial rec auto-mark (webcam)
- ✅ QR scan
- ✅ Login + email alerts
- ✅ CSV export
- ✅ JUnit tests
